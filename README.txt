This is my stop watch web application.

Technologies used:
- Node js
- Express

Project structure:
|-- stopwatch
|   |-- public
|        |-- client.js (front end javascript)
|        |-- index.html (front end html)
|   |-- server.js (express web server listening on 8080)
|__ |-- package.json (dependencies)

To run server:
1. node server.js

To view web application:
1. go to http://localhost:8080
